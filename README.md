# 🎨 Aplicando CSS

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar a aplicação de **CSS em páginas HTML**, focando na melhoria visual, organização do conteúdo e separação entre estrutura e estilo.

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3

## 🎯 Funcionalidades

* Estilização de elementos HTML
* Aplicação de cores, tipografia e espaçamento
* Organização visual da página
* Separação entre conteúdo (HTML) e apresentação (CSS)
